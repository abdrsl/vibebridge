# 飞书机器人私信配置指南

## 问题：机器人私信没有输入框

当用户在飞书中打开机器人私信时，看不到消息输入框，无法发送消息。

## 原因

飞书机器人默认只能在群聊中使用，要支持私信（单聊）需要额外配置：

1. **未开启"机器人"能力**
2. **未配置"单聊"权限**
3. **机器人未发布到企业**

---

## 解决步骤

### 步骤 1: 开启机器人能力

1. 登录 [飞书开放平台](https://open.feishu.cn/)
2. 进入你的应用 → **能力管理**
3. 找到 **机器人** 能力，点击 **开启**
   - 机器人名称：你的机器人名字
   - 机器人描述：简短描述
   - 消息卡片：建议开启
   - 保存

### 步骤 2: 配置单聊权限

1. 进入 **权限管理**
2. 搜索并添加以下权限：
   - ✅ `im:chat:readonly` - 读取会话信息
   - ✅ `im:message:send` - 发送消息
   - ✅ `im:message.group_msg` - 发送群组消息
   - ✅ `im:message.p2p_msg` - 发送单聊消息（**关键！**）

### 步骤 3: 配置事件订阅

1. 进入 **事件订阅**
2. 确保已配置 **请求地址 URL**：
   ```
   https://your-domain.com/feishu/webhook/opencode
   ```
3. 添加以下事件：
   - ✅ `im.message.receive_v1` - 接收消息

### 步骤 4: 发布机器人

1. 进入 **版本管理与发布**
2. 点击 **创建版本**
3. 填写版本信息：
   - 版本号：1.0.0
   - 更新说明：初始版本
   - 可用性状态：**已发布**
4. 点击 **保存并发布**
5. **申请发布** → 等待管理员审批

### 步骤 5: 添加到企业

发布后，需要企业管理员将机器人添加到企业：

**方法一：管理员后台添加**
1. 飞书管理后台 → 工作台 → 应用管理
2. 找到你的应用 → 添加到企业

**方法二：用户自行添加**
1. 飞书客户端 → 工作台 → 添加应用
2. 搜索你的机器人名称
3. 点击添加

---

## 验证配置

配置完成后，测试私信功能：

1. 在飞书搜索框搜索机器人名称
2. 点击进入机器人主页
3. 点击 **发消息** 按钮
4. 应该能看到消息输入框

## 常见问题

### Q1: 配置完成后还是没有输入框？

**检查清单：**
- [ ] 机器人能力已开启
- [ ] `im:message.p2p_msg` 权限已添加
- [ ] 应用已发布（不是草稿状态）
- [ ] 用户已添加机器人到企业
- [ ] 事件订阅 URL 可访问

### Q2: 群聊正常，私信不行？

这是典型的缺少 `im:message.p2p_msg` 权限问题。

**解决：**
1. 权限管理 → 添加 `im:message.p2p_msg`
2. 重新发布应用
3. 用户需要重新添加机器人

### Q3: 如何测试事件订阅是否正常？

使用飞书开放平台的 **调试工具**：

1. 进入 **事件订阅** 页面
2. 点击 **调试工具**
3. 选择事件类型：`im.message.receive_v1`
4. 填写测试数据，点击发送
5. 检查你的服务器是否收到请求

### Q4: 用户看不到机器人？

**可能原因：**
- 应用未发布
- 应用可见范围设置错误
- 用户不在应用的可见范围内

**解决：**
1. 进入 **应用功能 → 基础信息**
2. 检查 **可用范围**：
   - 选择 **全部成员** 或
   - 选择特定部门/用户
3. 重新发布
